# 34-keys layout (MacOS specific)

## Nuances
* tap-hold mods shifted a row down
* combos for ESC, TAB, ENTER, DELETE, BACKSPACE, OPT+BACKSPACE on both sides
* combos for CMD+C, CMD+X, CMD+V, CMD+Z, CMD+SHIFS+Z on left side
